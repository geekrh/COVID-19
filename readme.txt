Steps to install enviroments   : 



1) you have to install node js (my npm version  6.13.6)  : https://nodejs.org/en/download/ 

2) to install angular you have to open your termin and paste this command  : 
npm install -g @angular/cli@latest 

3) to install ionic you have to paste this command : npm install -g @ionic/cli

4) next step you have to clone our project from github : git clone https://github.com/geekrh/COVID-19.git 

5) to install dependecies tap : npm install 

6)finally you can run the project via (npm start / ng serve  / ionic serve )
